Test Page - Making Sure it Works

This is a sample guide designed to test:

  • Proper translation of +something+ to something
  • Automatic heading numbering
  • Generation of Table of Contents
  • Code blocks with filenames
  • Provide NOTE/INFO/WARNING shortcuts

1 - I'm the first heading!

This is the first paragraph. It's nice and short.

$ some bash code here
 ~ Yup...I think this worked...

Another paragraph goes here. I could type for a fairly long time and make some really long sentences and keep it going until the end of all days, but I think I shall defer until a better time. That makes much more sense.

some/one/line/code/here

Here is some more paragraph text.

1.1 - I'm a subheading.

I'm even more text.

Woah! Watch out for me.

1.1.1 - I'm a sub-subheading.

And here is some more text.

2 - I am the second heading. I'm not so great.

I'm a pretty cool note!

Woot, we got more text coming right at you!

2.1 - Another subheading.

With some more text.

I'm just here to tell you about something.

And here is some fancy code below:

/some/ruby/file.rb

a = 1 + 2
b = a / 3
c = a * b

puts a, b, c
2.1.1 - Another sub-subheading.

With even more text/code.

/i/am/some/javascript3.js

{
  here: {
    comes: "the bride",
    1: "more time",
    "something else": [3, 'b', { fu: 'bar' }]
  }
}
2.1.2 - Yet one more sub-subheading.

With the last bit of text.

Chapters

  1. I'm the first heading!

    • I'm a subheading.

  2. I am the second heading. I'm not so great.
